## Prerequisites

* Golang 1.16.x
* Docker 19.03+
* Docker Compose 1.25+

## Installation

```sh
docker-compose up -d 
```

## Used Technologies

* Golang 1.16.3
* Echo
* Echo Swagger
* Swag
* Template
* Go Uuid
* Go Urn  
* Validator
* Universal Translator


## Golang Swagger

> You can access the Golang Swagger from the following url.

http://localhost:9000/api
