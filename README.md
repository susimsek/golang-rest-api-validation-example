## Prerequisites

* Golang 1.16.x
* Docker 19.03+
* Docker Compose 1.25+

## Installation

```sh
docker-compose up -d 
```

## Used Technologies

* Golang 1.16.3
* Echo
* Echo Swagger
* Swag
* Template
* Air  
* Go Uuid
* Go Urn  
* Validator
* Universal Translator

## Development

Please note that it requires Go 1.16+ since I use `go mod` to manage dependencies.

```bash

# 1. install dependencies
go mod download

# 2. For less typing, you could add alias air='~/.air' to your .bashrc or .zshrc.
alias air='~/.air

# 3. run it 
air
```
